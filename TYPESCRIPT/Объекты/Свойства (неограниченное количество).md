# [`Свойства (неограниченное количество)`](../index.md/#объекты)

## Пример

### Через квадратные скобки

Ключ указывается в квадратных скобках. Имя может быть любым.

```ts
interface User {
  [key: string]: unknown;
}
```

### С помощью generic Record

```ts
const user: Record<string, unknown> = {};
```
