# [`Типизация объектов`](../index.md)

Объекты можно типизировать прямо после двоеточия или с помошью `interface` и `type`.

## Пример

```ts
const info: {
  officeId: number;
  isOpened: boolean;
  contacts: {
    phone: string;
    email: string;
    address: {
      city: string;
    };
  };
} = {
  officeId: 45,
  isOpened: false,
  contacts: {
    phone: '+79100000000',
    email: 'my@email.ru',
    address: {
      city: 'Moskva',
    },
  },
};
```
