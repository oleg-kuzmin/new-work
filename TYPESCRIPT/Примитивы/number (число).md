# [`number (число)`](../index.md/#примитивы)

Тип number. Для чисел с плавающей точкой отдельного типа нет.

## Пример

```ts
let a = 1; // : number
let b = -10; // : number
let c = 3.5; // : number
let d = 10_000; // : number
let e = NaN; // : number

let revenue: number = 1000;
let bonus: number = 500;
let res: number = revenue + bonus;
```
