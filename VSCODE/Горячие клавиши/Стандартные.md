# [`Стандартные`](../index.md)

```bash
Ctrl + Shift + p           # список команд
Ctrl + Shift + o           # список переменных
Ctrl + p                   # список файлов
Ctrl + pp                  # переключение на предыдущую вкладку
Ctrl + Tab                 # переключение по вкладкам
Ctrl + PageDown            # переключить на следующую вкладку
Ctrl + PageUp              # переключить на предыдущую вкладку
Ctrl + 0                   # фокус на проводнике
Ctrl + 1                   # фокус на редакторе
Ctrl + w                   # закрыть файл
Ctrl + k w                 # закрыть все файлы
F2                         # изменить имя переменной, в том числе в файле, импортах, экспортах
```

## Редактор

```bash
Tab                        # табуляция вперед
Shift + Tab                # табуляция назад
Alt + ЛКМ                  # мультикурсор
Alt + ↑                    # переместит строку вверх
Alt + ↓                    # переместит строку вниз
Alt + Shift + ↑            # скопировать строку вверх
Alt + Shift + ↓            # скопировать строку вниз
Alt + цифра                # переключение на вкладку
Ctrl + c                   # скопировать строку
Ctrl + x                   # вырезать строку
Ctrl + f                   # поиск
Ctrl + h                   # замена
Ctrl + Alt + Enter         # подтвердить замену
Ctrl + d                   # выделяет последовательно слово и устанавливает курсор
Ctrl + k Ctrl + d          # снимает курсор и идет дальше по поиску
Ctrl + F2                  # выделит все совпадения и можно поменять текст
Ctrl + z                   # отмена последнего действия
Ctrl + y                   # обратная операция
Ctrl + p                   # недавно открытые файлы
Ctrl + ↑                   # скролл вверх
Ctrl + ↓                   # скролл вниз
Ctrl + ←                   # перемещение по словам назад
Ctrl + →                   # перемещение по словам вперед
Ctrl + L                   # выделить строку
Ctrl + Space               # подсказки
Ctrl + Shift + .           # навигация по хлебным крошкам
Ctrl + Shift + k           # удалить строку
Ctrl + Shift + l           # выделить все элементы
Ctrl + Shift + ←           # выделение по словам назад
Ctrl + Shift + →           # выделение по словам вперед
Ctrl + Shift + \           # переход к закрывающей скобке
Ctrl + Shift + [           # свернуть блок кода
Ctrl + Shift + ]           # развернуть блок кода
Ctrl + Shift + PageDown    # переместить вкладку вперед
Ctrl + Shift + PageUp      # переместить вкладку назад
Ctrl + Tab                 # переключение на вкладку из списка
Shift + ↓                  # выделить строку
Shift + ↑                  # выделить строку
```

## Работа с ошибками

```bash
F8                         # переключить вперед
Shift + F8                 # переключить назад
```

## Расширение Better Comments

```bash
//#                        # желтый комментарий
//*                        # зеленый комментарий
//!                        # красный комментарий
//?                        # синий комментарий
//todo                     # оранжевый комментарий
////                       # серый перечеркнутный комментарий
```

## Расширение CSS Navigation

```bash
F12                        # переместить на класс css
Ctrl + ЛКМ                 # переместить на класс css
Alt + F12                  # вызвать окно совпадений css
```

## Пользовательские

```bash
Ctrl + n                   # Файл: создать файл
Ctrl + Shift + n           # Файл: создать папку
Ctrl + Shift + w           # Свернуть папки в проводнике
Ctrl + s                   # Форматировать документ
Ctrl + / (numpad)          # Закомментировать или раскомментировать строку
Ctrl + * (numpad)          # Закомментировать или раскомментировать блок
Ctrl + 1 (numpad)          # Live Server: Open with Live Server
Ctrl + 2 (numpad)          # Live Server: Stop Live Server
Ctrl + 3 (numpad)          # Emmet: Перенести с сокращением
Ctrl + 4 (numpad)          # Quokka.js: Start on Current File
Ctrl + 5 (numpad)          # Quokka.js: Stop All
Ctrl + 6 (numpad)          # Translate selection(s) to preffered language
Ctrl + 7 (numpad)          # Режим ZEN
Ctrl + 8 (numpad)          # Create New Templated Folder
Ctrl + 9 (numpad)          # Сортировка строк по возрастанию
```
