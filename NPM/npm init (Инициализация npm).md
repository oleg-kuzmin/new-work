# npm init (Инициализация npm)

```bash
# запустит инициализацию с рядом вопросов
npm init

# запустит инициализацию с автоматическими ответами на вопросы
npm init --yes

# вопросы:
# package name
# version
# description
# entry point
# test command
# git repository
# keywords
# author
# license

# после инициализации автоматически создаст файл package.json
```
