# [`console.table()`](../index.md)

Отображает массив объектов в виде таблицы.

## Синтаксис

```js
console.table(array);
```

### `array`

Массив - массив объектов.

## Пример

```js
const data = [
  { section: 'HTML', count: 212 },
  { section: 'CSS', count: 121 },
  { section: 'JavaScript', count: 182 },
  { section: 'Tools', count: 211 },
];

console.table(data);
```
