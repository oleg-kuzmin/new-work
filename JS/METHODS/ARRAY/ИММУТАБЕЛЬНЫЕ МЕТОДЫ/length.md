# `.length`

Массивы хранят свой размер в свойстве `length`.

## Синтаксис

```js
array.length;
```

## Возвращает

### `number`

Строка с значением количества элементов.

## Пример

```js
const infoArray = ['Россия', 'Москва', 144.5, 'Russian ruble', true];
console.log(infoArray.length); // "5"
```
