# `Array.from()`

Создаст массив из псевдомассива-коллекции.

## Cинтаксис

```js
Array.from(collection);
```

### `collection`

Коллекция - псевдомассив.

## Возвращает

### `array`

Созданный массив.

## Пример

```js
const posts = content.querySelectorAll('.post');
const postsArray = Array.from(posts);
```
