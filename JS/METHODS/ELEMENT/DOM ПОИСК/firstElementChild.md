# `.firstElementChild`

Свойство `firstElementChild` позволяет получить первый дочерний элемент.

Все эти свойства доступны только для чтения. Перезаписать их не получится.

## Синтаксис

```js
element.firstElementChild;
```

## Возвращает

### `element` || `null`

DOM-элемент. `null` - если не найдено.
