# `pageY`

Координаты по вертикали от угла веб-страницы. Пользоваться этими свойствами удобно, например, когда вы создаёте всплывающую подсказку около элемента, по которому кликнул пользователь.

## Синтаксис

```js
event.pageY;
```

## Возвращает

### `number`

Число пикселей.

## Пример

```js
event.pageY;
```
