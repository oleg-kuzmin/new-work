# `Math.floor()`

Округление до ближайшего целого в меньшую сторону.

## Синтаксис

```js
Math.floor(number);
```

### `number`

Число.

## Возвращает

### `number`

Число - округленное до целого в меньшую сторону.

## Пример

```js
console.log(Math.floor(15.52)); // 15
console.log(Math.floor(15.3)); // 15
console.log(Math.floor(-15.3)); // -16
```
