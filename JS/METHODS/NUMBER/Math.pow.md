# `Math.pow()`

Метод возвращает число, возведённое в степень.

## Синтаксис

```js
Math.pow(base, exponent);
```

### `base`

Число - основание.

### `exponent`

Число - степень, в которую возводится основание.

## Возвращает

### `number`

Число - возведённое в степень.

## Пример

```js
Math.pow(10, 3); // 1000
```
