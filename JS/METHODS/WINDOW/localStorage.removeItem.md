# `localStorage.removeItem()`

Удаляет запись из хранилища. Успешно выполнится даже если указанного ключа не существует в хранилище.

## Синтаксис

```js
window.localStorage.removeItem(key);
```

### `key (Строка)`

Строка - ключ.

## Возвращает

### `undefined`

## Пример

```js
window.localStorage.removeItem('name');
```
