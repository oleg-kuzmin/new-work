# [`evt.clientX`](../index.md)

Координаты по горизонтали от угла окна просмотра. Это подойдёт для позиционирования окна чата со специалистом, которое можно встретить на многих сайтах.

## Синтаксис

```js
evt.clientX;
```

## Возвращает

### `number`

Число пикселей.

## Пример

```js
evt.clientX;
```
