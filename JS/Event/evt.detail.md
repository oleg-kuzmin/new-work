# [`evt.detail`](../index.md)

Свойство возвращает подробную информацию о событии.

Свойство доступно только для чтения.

## Синтаксис

```js
evt.detail;
```

## Возвращает

### `number` (для `click` `dblclick`)

Количество кликов.

### `number` (для `mousedown` и `mouseup`)

Количество кликов + 1.
