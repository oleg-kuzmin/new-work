# Экспорт и импорт CommonJS

Можно использовать новый синтаксис ES Modules, если файл будет с расширением `.mjs`.

## По умолчанию

```js
// Экспорт по умолчанию
module.exports = {
  y: 1,
};

// Импорт по умолчанию
const y = require('./example.js');
```

## Именованный

```js
// Именованный импорт
const { y } = require('./example.js');

// Именованный экспорт
exports.y = 12;
```
