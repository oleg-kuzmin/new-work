# Приватные поля

Для объявления приватного свойства достаточно поставить перед его названием символ `#`.

Все приватные поля необходимо объявить заранее, до конструктора.

## Пример

```js
class Song {
  #genre;

  constructor(name, artist) {
    this.name = name;
    this.artist = artist;
    this.isLiked = false;
    this.#genre = 'Rock';
  }

  like() {
    this.isLiked = !this.isLiked;
  }

  #getGenre() {
    return this.#genre;
  }
}
```
