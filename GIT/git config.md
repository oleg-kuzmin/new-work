# [`git config`](./index.md)

Установка глобальных настроек.

## Синтаксис

```bash
git config --global user.name "Oleg Kuzmin"           # Установка имени.
git config --global user.email "info@oleg-kuzmin.ru"  # Установка почты.
git config --list                                     # Проверка всех свойств конфига.
```
