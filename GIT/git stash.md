# [`git stash`](./index.md)

Позволяет спрятать изменения.

## Синтаксис

```bash
git stash                           # спрятать незакоммиченные изменения
git stash save "Changed index.html" # назвать спрятанные незакоммиченные изменения
git stash list                      # показать все спрятанные изменения и в каких ветках они сделаны
git stash clear                     # очистить список всех спрятанных изменений
git stash apply stash @{n}          # вернуть только нужные изменения, n - номер в stash list
git stash pop                       # вернуть последние спрятанные изменения в любой ветке
```
