# Горячие клавиши Git Bash

```bash
Tab                        // автозаполнение
Ctrl + A                   // переход в начало строки
Ctrl + E                   // возврат в конец строки
Ctrl + L                   // очищает экран от текста, аналог команды clear
↑                          // обращение к последней введённой команде
↓                          // обращение к предпоследней введённой команде
Q                          // выйти
Shift + ZZ (:wq)           // Выйти из файла сохранив изменения (командный режим)
Shift + ZQ (:q!)           // Выйти из файла без сохранения изменений (командный режим)
```
