# [`find`](./index.md)

Команда для поиска файлов.

```bash
# Синтаксис.
find директория ключ элемент

# Ищет в текущей директории конкретный файл.
find . -name 'index.html'

# Ищет в текущей директории файлы с именем index.
find . -name 'index'

# Ищет в текущей директории файлы, начинающиеся на index.
find . -name 'index*'

# Ищет в текущей директории файлы больше 100 МБ.
find . -size +100M -print

# Ищет в текущей windows.
find C:/windows -name 'index'
```
