# [`Установка`](../index.md)

```bash
# Установка ядра Redux
npm i redux

# Установка React Redux
npm i react-redux

# Установка Redux Persist
# По умолчанию входит в rtk
npm i redux-persist

# Установка Redux Thunk
npm i redux-thunk

# Установка Redux Toolkit
# Содержит внутри ядро redux
# Поэтому нужно установить только rtk и react-redux
npm i @reduxjs/toolkit react-redux
```
