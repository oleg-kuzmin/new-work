# [`border-top-right-radius`](../index.md)

Можно управлять радиусом закругления каждого угла в отдельности.

## Пример

```css
.element {
  border-top-right-radius: 5px;
}
```

## Значения

### `border-top-right-radius: 0` (по умолчанию)

Закругления нет.

### `border-top-right-radius: 5px`

Радиус в пикселях.
