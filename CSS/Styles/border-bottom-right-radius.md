# [`border-bottom-right-radius`](../index.md)

Можно управлять радиусом закругления каждого угла в отдельности.

## Пример

```css
.element {
  border-bottom-right-radius: 5px;
}
```

## Значения

### `border-bottom-right-radius: 0` (по умолчанию)

Закругления нет.

### `border-bottom-right-radius: 5px`

Радиус в пикселях.
