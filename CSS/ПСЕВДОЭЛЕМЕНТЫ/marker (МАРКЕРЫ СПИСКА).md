# marker (Маркеры списка)

`::marker` — псевдоэлемент, отвечающий за маркерное поле. В нём находятся, например, маркеры списка.

## Пример

```html
<ul>
  <li>Цвет точек</li>
  <li>этого ненумерованного списка</li>
  <li>#2e9aff</li>
</ul>
```

```css
li::marker {
  color: #2e9aff;
}
```

## Результат

<style>
  li::marker {
    color: #2e9aff;
  }
</style>

<ul>
  <li>Цвет точек</li>
</ul>
