# [`Соседние селекторы`](../index.md)

## Синтаксис

### `селектор + селектор`

## Пример

```html
<div class="block">
  <ul>
    <li class="item">1</li>
    <li class="item">2</li>
    <li class="item">3</li>
  <ul>
</div>
```

```scss
// выберет элемент <li>, который следует СРАЗУ после соседнего элемента <li>
li + li {
  font-size: 14px;
}
```
