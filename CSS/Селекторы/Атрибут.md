# [`0.1.0. Атрибут`](../index.md)

## Пример

```scss
// булевый атрибут (0.1.1)
input[checked] {
  font-size: 14px;
}

// значение атрибута в точности равно строке (0.1.1)
a[href='www.yandex.ru'] {
  font-size: 14px;
}

// значение атрибута в точности равно строке или содержит строку в своём (разделённом пробелами) списке значений (0.1.1)
a[href~='www'] {
  font-size: 14px;
}

// значение атрибута в точности равно строке или начинается со строки, за которым сразу следует дефис (0.1.1)
a[href|='.pdf'] {
  font-size: 14px;
}

// значение атрибута начинается с value (0.1.1)
a[href^='.pdf'] {
  font-size: 14px;
}

// значение атрибута заканчивается на value (0.1.1)
a[href$='.pdf'] {
  font-size: 14px;
}

// значение атрибута содержит value, независимо от его положения внутри строки (0.1.1)
a[href*='.pdf'] {
  font-size: 14px;
}
```
