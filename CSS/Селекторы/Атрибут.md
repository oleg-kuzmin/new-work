# [`0.1.0. Атрибут`](../index.md)

## Пример

```scss
// булевый атрибут
input[checked] {
  font-size: 14px;
}

// значение атрибута в точности равно строке
a[href='www.yandex.ru'] {
  font-size: 14px;
}

// значение атрибута в точности равно строке или содержит строку в своём (разделённом пробелами) списке значений
a[href~='www'] {
  font-size: 14px;
}

// значение атрибута в точности равно строке или начинается со строки, за которым сразу следует дефис
a[href|='.pdf'] {
  font-size: 14px;
}

// значение атрибута начинается с value
a[href^='.pdf'] {
  font-size: 14px;
}

// значение атрибута заканчивается на value
a[href$='.pdf'] {
  font-size: 14px;
}

// значение атрибута содержит value, независимо от его положения внутри строки
a[href*='.pdf'] {
  font-size: 14px;
}
```
