# Вложенность селекторов

Используется оператор наследования `&`

### `h2 span`

```css
h2 {
  & span {
    color: var(--color);
  }
}
```

### `h2 + span`

```css
h2 {
  & + span {
    color: var(--color);
  }
}
```

### `h2:has(span)`

```css
h2 {
  &:has(span) {
    color: var(--color);
  }
}
```
