# `border-top-left-radius` (Скругление границы сверху слева)

Можно управлять радиусом закругления каждого угла в отдельности.

## Пример

```css
.element {
  border-top-left-radius: 5px;
}
```

## Значения

### `border-top-left-radius: 0` (по умолчанию)

Закругления нет.

### `border-top-left-radius: 5px`

Радиус в пикселях.
