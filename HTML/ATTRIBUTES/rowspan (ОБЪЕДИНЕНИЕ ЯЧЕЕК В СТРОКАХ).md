# `rowspan` (Объединение ячеек в строках)

Нужен для объединения ячеек из 2 или более строк.

## Применяется к тегам

- [`<th> (ЗАГОЛОВОК)`](<../TAGS TABLE/th (ЗАГОЛОВОК).md>)
- [`<td> (ЯЧЕЙКА)`](<../TAGS TABLE/td (ЯЧЕЙКА).md>)

## Пример

```html
<th rowspan="2">iPhone</th>
<td rowspan="2">iPhone</td>
```
