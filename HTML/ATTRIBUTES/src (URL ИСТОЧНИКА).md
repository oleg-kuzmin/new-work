# `src` (Url-адрес источника)

Обязательный атрибут. В качестве значения указывается адрес картинки. Абсолютная или относительная ссылка до файла в любом допустимом формате.

## Применяется к тегам:

- [`<img>`](<../TAGS IMAGE/img (ИЗОБРАЖЕНИЕ).md>)

## Пример

```html
<!-- <img> -->
<img src="logo.png" />
```
