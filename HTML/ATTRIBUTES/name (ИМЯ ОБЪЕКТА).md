# `name` (Имя объекта)

Имя для связи с JS.

## Применяется к тегам:

- [`<object>`](<../TAGS MEDIA/object (ВСТРАИВАНИЕ МЕДИА).md>)

## Пример

<object name='pdf-example' type="application/pdf" data="example.pdf" width="600" height="700">
  <a href="example.pdf">Лунная соната № 14 PDF</a>
</object>
