# `reversed`

_(reversed: перевернутый)_

Атрибут reversed меняет направление нумерации на противоположное (только для `<ol>`). Этот атрибут не требует значения.

## Применяется к тегам

- [`<ol>`](../../TAGS/BLOCK/ol.md) упорядоченный список

## Пример

```html
<ol reversed>
  <li>Белка</li>
  <li>Лиса</li>
  <li>Ёж</li>
</ol>
```
