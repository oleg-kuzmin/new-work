# Создание компонента

По умолчанию конструктор можно не писать.

## Пример

```jsx
import React from 'react';

class UserGreeting extends React.Component {
  // constructor(props) {
  //   super(props);
  // }
  render() {
    return <h1>Здравствуйте, {this.props.fullName}</h1>;
  }
}

export default UserGreeting;
```
