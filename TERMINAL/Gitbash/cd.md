# `cd`

Сменить каталог.

```bash
# Синтаксис.
cd имя_папки

# Перейти на уровень выше (назад).
cd ..

# Перейти в домашнюю директорию.
cd ~

# Перейти на рабочий стол.
cd ~/Desktop/

# Если в названии папки есть пробелы, используйте кавычки.
cd "Рабочий стол"

# Перейди в папку html, находящуюся в папке first-project.
cd first-project/html
```
