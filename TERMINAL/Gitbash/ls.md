# `ls`

_(list directory contents: вывести список содержимого каталога)_

Посмотреть содержимое каталога.

```bash
# Показать файлы.
ls

# Показать файлы.
ls C:/...

# Показать файлы, в том числе скрытые.
ls -a

# Показать файлы подробно.
ls -l

# Показать файлы подробно, в том числе скрытые.
ls -la
```
