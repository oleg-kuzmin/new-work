# [`git push`](../index.md)

```bash
# "origin main" - название главной ветки удаленного репозитория.
# Команда передает изменения из локального репозитория в удаленный.
# Устанавливает восходящие отношения отслеживания.
# Это позволяет вам извлекать или отправлять будущие изменения максимально просто 
# с помощью git pull или git push без указания удаленного устройства или ветки.
git push -u origin main

# Отправляем все изменения в удалённый репозиторий.
git push
```
